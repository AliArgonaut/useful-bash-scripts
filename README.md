# A Collection of Useful Bash and Python Scripts
1. vita-mix : generate opinionated vite projects with tailwind and shadcn
2. Dot-Up-Pckgs : fetches updated AUR and Pacman files and puts them in my dotfiles directory for git tracking and easy reinstalls
3. Spring-Jar : builds and runs maven and gradle projects give the root directory of a module
4. html-to-pdf : combines a variety of tool options for creating pdf files from markdown files
